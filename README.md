# AI Interview Coach

AI Interview Coach is a GenAI-powered mock interview platform that simulates domain-specific technical interviews and evaluates responses using NLP.

## Features

- AI-generated interview questions
- Sentiment analysis of answers
- Confidence scoring
- Real-time feedback dashboard

## Tech Stack

- Python
- Streamlit
- NLP (TextBlob)
- Scikit-learn
- AWS Bedrock / Gemini API (optional integration)

## Installation

```bash
git clone https://github.com/yourusername/ai-interview-coach.git
cd ai-interview-coach
pip install -r requirements.txt
streamlit run app.py
